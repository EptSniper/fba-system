How to run it

The scout is a Python agent in your scout/ folder. It needs a paid Keepa key (set it in Settings). Then on your computer:

cd "Amazon FBA\scout"
pip install -r requirements.txt
# paste your KEEPA_KEY into .env
python run_scout.py --once --dry-run

One find + score run, prints the picks. Add a Discord webhook for alerts; drop --dry-run to post live.
