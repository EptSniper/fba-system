# Assets

Screenshots and files Mehmet shares get saved here and indexed in
[`../tracking/links-and-assets.md`](../tracking/links-and-assets.md).

**Naming:** `YYYY-MM-DD_topic.ext` (e.g. `2026-06-19_startup-tools-stack.md`).

**Note on chat images:** images pasted into chat aren't always written to disk as
files automatically. When the original binary isn't available, Claude saves a
faithful **text transcription** (`.md`) here so the content is still usable by the
AI, and notes that the original image lives in the chat on that date. If Mehmet
later re-uploads the file in a way that lands on disk, the original is added too.
